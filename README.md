# AI-Based Emotion Detector

## Project Description
The **Emotion Detector** is a web-based application developed using **Python**, **Flask**, and the **Watson NLP library**. This application takes user-provided text as input and analyzes it to detect various emotions, including anger, disgust, fear, joy, and sadness. It also identifies the "dominant emotion" based on the highest score returned by the NLP model.

## Features
* **NLP Integration**: Utilizes the Watson NLP service for precise emotion analysis.
* **Web Interface**: A clean UI built with HTML/CSS and served via Flask.
* **Error Handling**: Gracefully handles blank or invalid inputs with descriptive feedback.
* **Unit Testing**: Verified logic using Python's `unittest` framework.
* **Static Analysis**: Compliant with PEP 8 standards, achieving a perfect 10/10 Pylint score.
